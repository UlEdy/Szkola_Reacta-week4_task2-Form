# Form

This project consists of a form with inputs like name, email, and bio, a select for gender, and a checkbox for terms. All inputs are shown in a summary. This task is from the Szko≈Ça Reacta course.

## Getting Started

To start the project, follow these steps:

1. Open a terminal.
2. Type the following command to install dependencies: `npm install`.
3. Once the dependencies are installed, run the project using: `npm start`.
